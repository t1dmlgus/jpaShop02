# JPA-Shop


## 기능 요구 사항

---
- 회원은 상품을 주문할 수 있다.
- 주문 시 여러 종류의 상품을 선택할 수 있다.
- 상품의 종류는 도서, 음반, 영화가 있다.
- 상품을 카테고리로 구분할 수 있다.
- 상품 주문 시 배송 정보를 입력할 수 있다.

<br>

## 구현 기능 목록

---
- 회원 기능
    - 회원 등록
      - 중복 회원 예외처리
    - 회원 조회

- 상품 기능
    - 상품 등록
    - 상품 수정
    - 상품 조회

- 주문 기능
    - 상품 주문
    - 주문 내역 조회
    - 주문 취소
